Interacting Test Suites
^^^^^
Definition:

* A special case of Interacting Tests where the tests are in different test suites.


Code Example:

.. code-block::

  Suite1.run()--> Green
  Suite2.run()--> Green
  Suite(Suite1,Suite2).run()--> Test C in Suite2 fails
