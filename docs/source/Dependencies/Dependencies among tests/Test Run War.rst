Test Run War
^^^^^
**Definition:**

* Test failures occur at random when several people are running tests simultaneously.


**Code Example:**

.. code-block:: ruby

  Suite.run() --> Test 3 fails
  Suite.run() --> Test 3 crashes
  Suite.run() --> All tests pass
  Suite.run() --> Test 3 fails
