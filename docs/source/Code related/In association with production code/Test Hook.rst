Test Hook
^^^^^
Definition:

* Conditional logic within the SUT determines whether the "real" code or test-specific logic is run.


Code Example:

.. code-block:: ruby

  class ApplicationController < ActionController::Base
    unless Rails.env.test?
      before_filter :require_login
    end
  end
