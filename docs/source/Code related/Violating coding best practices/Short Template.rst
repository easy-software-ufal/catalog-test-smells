Short Template
^^^^^
Definition:

* Template definition is very short (in terms of characters or number of fields).


Code Example:

.. code-block:: pseudo

  template integer exampleTemplate := 1;

  testcase exampleTestCase() runs on ExampleComponent {
    // ...
    pt.send(exampleTemplate);
    // ...
  }
