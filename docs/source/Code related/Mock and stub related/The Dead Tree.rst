The Dead Tree
^^^^^
Definition:

* A test which where a stub was created, but the test wasn't actually written.

Also Known As:

* Process Compliance Backdoor

Code Example:

.. code-block:: typescript

  class TD_SomeClass {
    public void testAdd() {
      assertEquals(1+1, 2);
    }
  }
